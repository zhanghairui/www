================
	代码规范
================

CSS、HTML和JS命名
	· 命名不允许使用拼音
	· CSS和HTML命名，多个单词间使用。例如： - 连接 css-demo
	· JS命名使用驼峰命名，例如：demoJs
	· 命名最多不超过三个单词

主体：	main      
外层：	wrap
功能条：func-bar     
友情链接：friend-link
版权：	copyright
页眉：	header
页脚：	footer
标题：	title
主导航：main-nav
子导航：sub-nav
菜单：	menu

搜索：	search
